# 选择器

### 选择器的优先级

`!important` > 行内样式 > `#id` > `.class` > 标签 > *（全局） > 继承 > 默认 

注：有个css选择器优先级的算法，可以看下

--- 

### 伪元素和伪类的区别

伪类: 通常以:开头，比如：
- input的 `:foucs`, `:hover`, `:active`等
- 选择器的 `:first-child, `:nth-child(n)`等

伪元素: 通常以::开头，里面有content属性，比如：
- `::before`， `::after`
