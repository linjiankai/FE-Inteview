# 组件通信

### 父组件=>子组件通信

使用 props

```js
    //父组件
    <child data="数据"></child>
    //子组件
    <div>{this.props.data}</div>
```

### 子组件=>父组件通信

```js
    //父组件
    <child onToParent={}></child>
    //子组件
    <div onClick={this.props.onToParent()}>数据</div>
```
