# 性能优化

### react代码上性能优化
- key 
- shouldComponentUpdate
- PureComponent
- render里面尽量减少新建变量和bind函数，props减少参数
- 使用Immutable.js
- useCallback
- useMemo,React.Memo
- Redux中reselect
